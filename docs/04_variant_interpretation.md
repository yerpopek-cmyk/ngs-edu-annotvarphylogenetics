# 4. Интерпретация вариантов

## 4.1. Лестница интерпретации

Интерпретация вариантов — это многоуровневый процесс оценки биологической и клинической значимости найденных изменений.

```
Molecular consequence → Transcript context → Population frequency → Known disease links → Regulatory & cell-state → Actionability
```

| Уровень | Описание | Инструменты/Базы |
|---------|----------|------------------|
| **1. Molecular consequence** | SO-термины: missense, frameshift, splice | VEP, SnpEff, bcftools csq |
| **2. Transcript context** | Изоформа, экзон, CDS, UTR | Ensembl, RefSeq |
| **3. Population frequency** | Частота в популяциях, этнический контекст | gnomAD, 1000 Genomes |
| **4. Known disease links** | Связь с заболеваниями | ClinVar, OMIM, COSMIC |
| **5. Regulatory & cell-state** | Энхансеры, сплайсинг, экспрессия | Enformer, AlphaGenome |
| **6. Actionability** | Клиническая значимость, терапия | CIViC, OncoKB, ClinGen |

## 4.3. Стандарты ACMG/AMP

### Классификация вариантов

Американский колледж медицинской генетики и геномики (ACMG) разработал стандарты классификации вариантов:

| Класс | Обозначение | Описание | Вероятность патогенности |
|-------|-------------|----------|-------------------------|
| **5** | ⭐⭐⭐⭐⭐ **Pathogenic** | Патогенный | >99% |
| **4** | ⭐⭐⭐⭐ **Likely pathogenic** | Вероятно патогенный | 90–99% |
| **3** | ⭐⭐⭐ **VUS** | Вариант неопределённой значимости | 10–89% |
| **2** | ⭐⭐ **Likely benign** | Вероятно безвредный | 1–9% |
| **1** | ⭐ **Benign** | Безвредный | <1% |

### Типы доказательств (Evidence types)

#### Очень сильные (PVS1)
- **PVS1**: Null variant (nonsense, frameshift, canonical ±1,2 splice sites) в гене, где loss of function является известным механизмом заболевания

#### Сильные (PS1–PS4)
- **PS1**: Тот же аминокислотный изменение, что и ранее установленный патогенный
- **PS2**: De novo в пациенте с заболеванием, подтверждённое родительство
- **PS3**: Хорошо установленная функциональная вредность
- **PS4**: Значительно повышенная частота в случаях vs контроле

#### Умеренные (PM1–PM6)
- **PM1**: Расположен в мутационном hotspot / критическом домене
- **PM2**: Отсутствует в контрольных популяциях (gnomAD)
- **PM3**: В транс-конфигурации с патогенным вариантом
- **PM4**: Изменение длины белка (in-frame indel)
- **PM5**: Другой missense в том же кодоне
- **PM6**: Предполагаемый de novo без подтверждения родительства

#### Слабые (PP1–PP5)
- **PP1**: Ко-сегрегация с заболеванием в семье
- **PP2**: Missense в гене, где missense являются частым механизмом
- **PP3**: Множественные computational предсказания вреда
- **PP4**: Фенотип специфичен для данного гена
- **PP5**: Репутабельный источник сообщил как патогенный

### Бенчмарк-калькуляторы ACMG

| Инструмент | Описание |
|------------|----------|
| **InterVar** | Автоматическая классификация по ACMG |

## 4.4. Современные инструменты интерпретации

### DeepVariant 🧠

**DeepVariant** — DL-коллер для SNV/indel, обучаемый на pileup-паттернах.

- Использует свёрточные нейронные сети (CNN)
- Преобразует BAM в изображения pileup
- Классифицирует генотипы как изображения
- Достигает state-of-the-art точности

```bash
run_deepvariant \
  --model_type=WGS \
  --ref=reference.fasta \
  --reads=sample.bam \
  --output_vcf=variants.vcf \
  --num_shards=8
```

### AlphaGenome 🔮

**AlphaGenome** — модель предсказания эффекта варианта на регуляцию/сплайсинг по контексту до ~1 Мб.

- Предсказывает влияние на эпигенетические метки
- Учитывает длинный контекст последовательности
- Моделирует эффекты distal регуляторных элементов

### Enformer 🧬

**Enformer** — transformer-based модель для предсказания:
- Профилей гистоновых модификаций
- Доступности хроматина (ATAC-seq)
- Уровней экспрессии генов
- Эффектов вариантов на регуляцию

## 4.5. Практический пайплайн интерпретации

```bash
#!/bin/bash
VCF="sample.filt.vcf.gz"
REF="hg38.fa"
GFF="genes.gff3.gz"
SAMPLE="S1"

# 1. Функциональная аннотация (bcftools csq)
bcftools csq -f $REF -g $GFF $VCF -Ov -o $SAMPLE.annotated.vcf

# 2. Добавление частот из gnomAD
bcftools annotate -a gnomad.vcf.gz -c INFO/AF $SAMPLE.annotated.vcf -Ov -o $SAMPLE.af.vcf

# 3. Предсказание патогенности (CADD scores)
# Загрузить через VEP или добавить из预计算ленных файлов

# 4. Фильтрация по редкости (AF < 0.01)
bcftools filter -i 'INFO/AF<0.01 || INFO/AF="."' $SAMPLE.af.vcf -Oz -o $SAMPLE.rare.vcf

# 5. Приоритизация по последствию
bcftools view -i 'INFO/Consequence="missense_variant" || INFO/Consequence="stop_gained"' $SAMPLE.rare.vcf

# 6. Проверка ClinVar
bcftools annotate -a clinvar.vcf.gz -c INFO/CLNSIG $SAMPLE.rare.vcf -Ov -o $SAMPLE.clinvar.vcf

# 7. Генерация отчёта
python scripts/generate_report.py --input $SAMPLE.clinvar.vcf --output $SAMPLE.report.html
```

## 4.7. Особые случаи

### Mosaicism 🧩

- Вариант присутствует только в части клеток
- Требуется специальная настройка variant caller
- Аллельная фракция может быть низкой (<10%)

### Compound heterozygosity 👥

- Два разных варианта в одном гене (по одному от каждого родителя)
- Важно учитывать фазу вариантов (trans-конфигурация)
- Требует семейного трио-анализа или long-read sequencing

### Structural variants in interpretation 🔍

- SV сложнее интерпретировать, чем малые варианты
- Требуется оценка overlap с генами и регуляторными элементами
- Базы данных: gnomAD-SV, DGV, ClinVar SV
