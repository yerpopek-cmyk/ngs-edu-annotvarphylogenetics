# 📐 Сводка формул (Formulas Reference)

## Полная справка по всем формулам NGS анализа

---

## 1. Аннотация генома

### E-value (статистическая значимость выравнивания)
```
E = m × n × 2^(-S)
```
| Переменная | Описание |
|------------|----------|
| `m` | Длина запроса (query length) |
| `n` | Размер базы данных |
| `S` | Score выравнивания (биты) |

**Интерпретация:** Меньшее E-value указывает на более значимое совпадение. E < 0.001 обычно считается значимым.

---

## 2. Вызов вариантов

### QUAL (Quality Score)
```
QUAL = -10 × log₁₀(P(error))
```
| Переменная | Описание |
|------------|----------|
| `P(error)` | Вероятность ошибки вызова варианта |

**Примеры:**
| QUAL | P(error) | Точность |
|------|----------|----------|
| 10 | 0.1 | 90% |
| 20 | 0.01 | 99% |
| 30 | 0.001 | 99.9% |
| 40 | 0.0001 | 99.99% |

### PL (Phred-scaled Likelihoods)
```
PL(g) = -10 × log₁₀(L(g))
```
| Переменная | Описание |
|------------|----------|
| `L(g)` | Правдоподобие генотипа `g` |

### GQ (Genotype Quality)
```
GQ = min(PL(other_genotypes)) - PL(best_genotype)
```

### Phred Score (общая формула)
```
Q = -10 × log₁₀(P)
```
где `P` — вероятность ошибки.

## 3. Структурные варианты

### Размер делеции
```
SV_size = observed_insert_size - expected_insert_size
```

### Поддержка split-read
```
Support = N(split_reads) / N(total_reads_at_locus)
```

### Чувствительность обнаружения SV
```
Sensitivity = TP / (TP + FN)
```
| Переменная | Описание |
|------------|----------|
| `TP` | Истинно положительные SV |
| `FN` | Ложно отрицательные SV |

### Точность обнаружения SV
```
Precision = TP / (TP + FP)
```
| Переменная | Описание |
|------------|----------|
| `FP` | Ложно положительные SV |

## 4. Интерпретация вариантов

### Частота аллелей (Allele Frequency)
```
AF = N(alt_alleles) / N(total_alleles)
```
| Переменная | Описание |
|------------|----------|
| `N(alt_alleles)` | Количество альтернативных аллелей |
| `N(total_alleles)` | Общее количество аллелей в популяции |

### Гетерозиготность (Heterozygosity)
```
H = 1 - Σ(pᵢ²)
```
| Переменная | Описание |
|------------|----------|
| `pᵢ` | Частота i-го аллеля |

### Байесовская оценка патогенности
```
Posterior_odds = Prior_odds × LR(ACMG_evidence)
```
| Переменная | Описание |
|------------|----------|
| `Prior_odds` | Априорные шансы патогенности |
| `LR` | Likelihood ratio свидетельств ACMG |

### Конверсия odds в вероятность
```
P = odds / (1 + odds)
```

## 5. Филогенетика

### Расстояние NJ (Neighbor-Joining)
```
D(i,j) = d(i,j) - (r(i) + r(j)) / (N-2)
```
| Переменная | Описание |
|------------|----------|
| `d(i,j)` | Исходное расстояние между таксонами i и j |
| `r(i)` | Σ d(i,k) — сумма расстояний от i до всех других |
| `N` | Число таксонов |

### Поправка Джукса-Кантора
```
d = -(3/4) × ln(1 - (4/3) × p)
```
| Переменная | Описание |
|------------|----------|
| `p` | Доля различающихся сайтов |
| `ln` | Натуральный логарифм |

### Поправка Кимуры 2-parameter (K80)
```
d = -(1/2) × ln(1 - 2P - Q) - (1/4) × ln(1 - 2Q)
```
| Переменная | Описание |
|------------|----------|
| `P` | Доля транзиций |
| `Q` | Доля трансверсий |

### Формула Байеса для филогенетики
```
P(Tree | Data) = [P(Data | Tree) × P(Tree)] / P(Data)
```
| Переменная | Описание |
|------------|----------|
| `P(Tree | Data)` | Апостериорная вероятность дерева |
| `P(Data | Tree)` | Правдоподобие данных при данном дереве |
| `P(Tree)` | Априорная вероятность дерева |
| `P(Data)` | Маргинальная вероятность данных |

### Bootstrap поддержка
```
Bootstrap_support = N(clade_in_replicates) / N(total_replicates) × 100%
```

### Модель HKY85 (расстояние)
```
d = -β × ln(1 - P/α - Q/β) - (1/2) × ln(1 - 2Q/β)
```
где:
- `α = 1/2 × (π_A×π_G + π_C×π_T) / (π_A+π_G)(π_C+π_T)`
- `β = 1/2 × (π_A×π_C + π_G×π_T + π_A×π_T + π_G×π_C)`
- `π_X` — частота нуклеотида X

## 6. Общие метрики NGS

### Покрытие (Coverage/Depth)
```
Depth = N(reads) × L(read_length) / L(genome_size)
```
| Переменная | Описание |
|------------|----------|
| `N(reads)` | Количество ридов |
| `L(read_length)` | Средняя длина рида |
| `L(genome_size)` | Размер генома |

### Ширина покрытия (Breadth of Coverage)
```
Breadth = L(covered_positions) / L(total_positions) × 100%
```

### Ошибка секвенирования (Sequencing Error Rate)
```
Error_rate = N(errors) / N(total_bases)
```

### Q-score для качества основания
```
Q = -10 × log₁₀(P_error)
```
| Q-score | P_error | Точность |
|---------|---------|----------|
| Q10 | 0.1 | 90% |
| Q20 | 0.01 | 99% |
| Q30 | 0.001 | 99.9% |
| Q40 | 0.0001 | 99.99% |

### GC-content
```
GC% = (N(G) + N(C)) / N(total_bases) × 100%
```

## 7. Статистические тесты

### Хи-квадрат для проверки равновесия Харди-Вайнберга
```
χ² = Σ((Oᵢ - Eᵢ)² / Eᵢ)
```
| Переменная | Описание |
|------------|----------|
| `Oᵢ` | Наблюдаемое количество генотипа i |
| `Eᵢ` | Ожидаемое количество генотипа i |

### Fisher's Exact Test для ассоциаций
```
p = (a+b)!(c+d)!(a+c)!(b+d)! / (a!b!c!d!n!)
```
для таблицы:
| | Cases | Controls |
|--|-------|----------|
| Variant | a | b |
| Ref | c | d |

## Быстрый справочник логарифмов

| Преобразование | Формула |
|----------------|---------|
| Phred → Probability | P = 10^(-Q/10) |
| Probability → Phred | Q = -10 × log₁₀(P) |
| Natural log → log₁₀ | log₁₀(x) = ln(x) / ln(10) |
